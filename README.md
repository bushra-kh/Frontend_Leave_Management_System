# Frontend Leave Management System

A lightweight frontend-only Leave Management System (LVMS) built with HTML, CSS, and JavaScript.

## What It Includes

- Employee dashboard with request summary
- Leave application form
- Leave history table with charts
- HOD approval portal
- HR final approval portal

## Project Files

- `index.html` - Overview and quick actions
- `leave_request.html` - Submit leave requests
- `history.html` - Request history and analytics
- `hod_db.html` - Department-level approvals
- `HR_db.html` - HR-level approvals
- `app.js` - Shared application logic and local storage handling
- `style.css` - Shared styling

## How to Run

1. Open `index.html` in your browser.
2. Navigate using the top menu.
3. Data is stored in browser local storage (no backend required).

## Notes

- This is a frontend assignment demo project.
- Clearing browser storage will remove saved leave records.
